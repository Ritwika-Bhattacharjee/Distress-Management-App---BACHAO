
# BACHAAO : Distress-Management-App

## Getting Started

### Open two terminals
1. Backend

```bash
# Install Dependencies
$ cd backend && npm install

# run the server
$ npm start
```

2. Frontend

```bash
# Install Dependencies
$ cd frontend && npm install

# run the application
$ npm run android
```

**NOTE :** put your machines ip_address as the apiURL in env..js
